# movies-data-analysis

## Описание проекта

  Данный проект посвящён анализу датасета с фильмами из топов Кинопоиска и IMDb.
**Цель проекта** — исследовать структуру данных, выявить закономерности, а также проанализировать влияние жанров, стран производства, бюджета и режиссёров на оценки и коммерческую успешность фильмов. 
  Проект реализован в формате анализа данных с использованием **Python** и библиотеки **pandas** и предназначен для демонстрации навыков работы с табличными данными, их преобразования, агрегации, визуализации и интерпретации результатов.

## Что включает в себя анализ?

В рамках проекта выполняются следующие действия:

1. Анализ структуры датасета
- загрузка CSV-файла с фильмами;
- базовое описание содержимого датасета.
- определение размера датасета (количество строк и признаков);
- анализ типов данных признаков;
- подсчет количества пропусков по каждому признаку

2. Анализ жанров: структура жанров и количество упоминаний в фильмах
- преобразование строкового признака Genres в список жанров;
- подсчёт количества упоминаний каждого жанра;
- определение количества уникальных жанров;
- выявление наиболее популярных жанров;
- анализ структуры жанров (ТОП-10 + категория «Прочее»);
- визуализация популярности и структуры жанров.

3. Анализ рейтингов: Сравнение рейтингов фильмов между Кинопоиском и IMDb
- исследование распределения рейтингов на обеих платформах;
- расчет разницы рейтингов между Кинопоиском и IMDb;
- выявление фильмов с наибольшими расхождениями в оценках;
- определение, на какой платформе рейтинг фильма выше чаще;
- анализ различий рейтингов в зависимости от страны производства;
- сравнение стран, для которых рейтинги выше на Кинопоиске и IMDb;
- визуализация результатов.

4. Исследование зависимости бюджета и рейтинга.
- построение диаграмм рассеяния;
- расчёт средних рейтингов по обеим платформам;
- анализ фильмов с наибольшими рейтингами и бюджетами;
- формирование вывода об отсутствии прямой зависимости бюджета и рейтинга.

5.Выявление наиболее успешных режиссеров
- расчет коммерческой успешности фильмов (кассовые сборы − бюджет);
- выявление наиболее коммерчески успешных режиссеров;
- определение самых окупаемых фильмов;
- визуализация результатов.

## Данные

### Источник данных
Используется CSV-файл с информацией о фильмах из топов Кинопоиска и IMDb из открытого источника.

### Размер датасета
477 строк x 14 столбцов

### Основные признаки
* Title - название фильма
* Year - год выпуска
* Genres - жанры фильма
* Country - страна(ы) производства
* Rating Kinopoisk - рейтинг Кинопоиска
* Rating Imdb - рейтинг IMDb
* Budget $ - бюджет фильма
* Fees $ - кассовые сборы
* Director - режиссер
* Age Limit - возрастное ограничение
* Description Kinopoisk - описание фильма из Кинопоиска на русском языке
* Description Imdb - описание фильма из Imdb на английском языке

## Требования

Для запуска проекта требуется:
- Python 3.9+
- Библиотека: **pandas** (указана в requirements.txt, используется в коде)

Проект выполнен в среде **Google Colab**, где необходимые библиотеки **уже установлены**.

## Инструкция

1. Скачать датасет "data_films.csv" на устройство.
2. Cкачать и Открыть файл "movies-data-analysis.ipynb" в Google Collab.
3. При необходимости обновить страницу, заново выполнить весь код, выбрав в начале файл "data_films.csv" на устройстве.
